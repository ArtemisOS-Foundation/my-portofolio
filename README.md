# my-portofolio

A personal portfolio site repository for ArtemisOS-Foundation. This project contains the source for a portfolio website showcasing projects, skills, and contact information.

## About

This repository hosts a personal portfolio to present projects, experience, and links. It is intended to be simple, responsive, and easy to maintain.

## Features

- Project showcase with descriptions and links
- Responsive layout for mobile and desktop
- Easy to update content

## Installation / Development

1. Clone the repository:

   git clone https://github.com/ArtemisOS-Foundation/my-portofolio.git

2. Open the project folder and serve the files (for example, use Live Server in VS Code or a simple HTTP server):

   - Using Python 3: `python -m http.server 8000`

3. Open http://localhost:8000 in your browser to view the site.
